# ShyTasks

A lightweight terminal-based task manager built in Python.

ShyTasks is the first software project released under **ShyLabs**, created as a learning project to practice software architecture, object-oriented programming, and version control with Git.

---

## Features

- ✅ Create tasks
- ✅ View tasks
- ✅ Delete tasks
- ✅ Priority selection menu
- ✅ Category selection menu
- ✅ Input validation
- ✅ Simple terminal interface

---

## Planned Features

- Save tasks to JSON
- Edit tasks
- Mark tasks as completed
- Search and filter tasks
- Task statistics
- Improved terminal interface
